# Changelog

All notable changes to Idle Rampage will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-01

### Added
- Initial release of Idle Rampage
- 6 unique building types with evolution tiers:
  - Scrap Works (5 tiers) - Resource generation
  - Weak Point Scanner (5 tiers) - Critical hit system
  - Training Facility (5 tiers) - Tap damage boost
  - Shield Generator (4 tiers) - Wave timer extension
  - Command Center (4 tiers) - Global production boost
  - Engineering Bay (5 tiers) - Upgrade cost reduction
- 100 waves of progressive difficulty
- 5 enemy tiers: Scrap Bots, Drones, Loaders, Mechs, AI Units
- Boss waves every 10 waves with bonus rewards
- Epic final boss gauntlet (waves 96-100)
- Prestige system with 10 permanent upgrades
- Worker assignment system with diminishing returns
- Lucky drop rewards from wave clears
- Offline earnings (up to 8 hours at 50% efficiency)
- Daily login rewards with streak bonuses
- Animated loading screen with gameplay tips
- Onboarding tutorial for new players
- Building storyline/lore system
- Auto-save every 30 seconds
- Import/export save data

### Technical
- React Native 0.82 with TypeScript
- Zustand state management
- react-native-reanimated for smooth animations
- Event-driven architecture with EventBus
- Centralized formula system for game balance
- Comprehensive test suite
